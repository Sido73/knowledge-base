# KnowledgeBase - Освітня платформа

**Курсовий проект** з дисципліни "Розробка веб-додатків за допомогою фреймворків"

## Опис
Веб-додаток для онлайн-освіти на React з функціоналом каталогу курсів, пошуку та системою "Обране".

## Технології
- React 19, React Router DOM
- Bootstrap 5, React-Bootstrap
- Vite, Git

## Запуск проекту
```bash
npm install
npm run dev
